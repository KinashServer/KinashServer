KinashScript is something like PHP but for KinashServer
It can get basic user information: 

URL:
KinashServer.getURL()
IP:
KinashServer.getIP()
CURRENT DATE:
Encoding.setUTF(true)
KinashServer.getDate()
REQUEST METHOD:
KinashServer.getRequestMethod()
FILE MIME:
KinashServer.getCurrentMime()
GET KINASHSERVER VERSION:
KinashServer.getVersion()
It also can: * return your custom errors: * Not working example: KinashServer[replace with dot]returnError(403) * Working example: KinashServer[replace with dot]returnError(403) * To test it you will need to replace [replace with dot] to . * It can return errors that are only listed in config * Or else it will be ignored
