<h1>KinashServer</h1>
KinashServer is simple Node.JS HTTP server with some basic config
<h2>Launch</h2>
<p>To launch, use these commands: 
<br>
<br>
npm install kinashserver
<br>
"C:\users\%username%\node_modules\kinashserver\kinashserver.cmd"</p>
