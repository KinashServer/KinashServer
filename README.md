<h1>KinashServer</h1>
KinashServer is simple node JS HTTP server with basic config
<h2>Launch</h2>
<p>To launch, use these commands: 
<br>
<br>
npm install kinashserver
<br>
"C:\users\%username%\node_modules\kinashserver\kinashserver.bat"</p>
